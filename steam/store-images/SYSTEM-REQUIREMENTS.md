# System Requirements — Paste into Steamworks

Go to: App Admin → Store Presence → Basic Info → System Requirements

---

## Windows — Minimum

OS: Windows 10 (64-bit)
Processor: Dual-core 1.5 GHz
Memory: 4 GB RAM
Graphics: WebGL 2.0 compatible (any GPU from 2014+)
Storage: 2 GB available space
Additional Notes: Requires Microsoft Edge WebView2 Runtime (included with Windows 10/11)

## Windows — Recommended

OS: Windows 11
Processor: Quad-core 2.0 GHz
Memory: 8 GB RAM
Graphics: NVIDIA GTX 660 / AMD RX 460 or better
Storage: 2 GB available space
Additional Notes: 1920x1080 display recommended

---

## macOS — Minimum

OS: macOS 12 Monterey
Processor: Apple M1 or Intel Core i3
Memory: 4 GB RAM
Graphics: Integrated (Metal-capable)
Storage: 2 GB available space

## macOS — Recommended

OS: macOS 14 Sonoma
Processor: Apple M1 or newer
Memory: 8 GB RAM
Graphics: Apple Silicon integrated
Storage: 2 GB available space
Additional Notes: 1920x1080 display recommended

---

## Linux / SteamOS — Minimum

OS: Ubuntu 20.04 / SteamOS 3.0
Processor: Dual-core 1.5 GHz
Memory: 4 GB RAM
Graphics: WebGL 2.0 compatible (Mesa 21.0+ or proprietary drivers)
Storage: 2 GB available space
Additional Notes: WebKitGTK 4.1+ required

## Linux / SteamOS — Recommended

OS: Ubuntu 22.04 / SteamOS 3.5
Processor: Quad-core 2.0 GHz
Memory: 8 GB RAM
Graphics: NVIDIA GTX 660 / AMD RX 460 or better with up-to-date drivers
Storage: 2 GB available space
Additional Notes: 1920x1080 display recommended
