# AR-59.5 — Three-Act Run Structure

## Overview

**Goal:** Compress the current 4-segment × 6-floor dungeon (24 floors, 72+ encounters, ~60-90 min runs) into a 3-act structure (~16-18 rooms per path, ~25-30 min runs). Keep STS-style branching maps within each act.

**Dependencies:** None at code level. Can be implemented independently.
- AR-59.13 (Enemy Roster) will define the specific enemies per act post-implementation.
- AR-59.7 (Boss Quiz Phases) will add quiz mechanics to Act 2/3 bosses post-implementation.

**Estimated complexity:** Medium — touches balance constants, map generator config, floor progression logic, game flow controller, and save state. No new systems; restructures existing ones.

---

## Context: Current Structure

The current codebase uses a 4-segment model:

| Segment | Floors | Region | Map Rows | Bosses |
|---------|--------|--------|----------|--------|
| 1 (Shallow Depths) | 1-6 | `shallow_depths` | 15 | floors 3 & 6 |
| 2 (Deep Caverns) | 7-12 | `deep_caverns` | 15 | floors 9 & 12 |
| 3 (The Abyss) | 13-18 | `the_abyss` | 15 | floors 15 & 18 |
| 4 (The Archive) | 19-24 | `the_archive` | 15 | floors 21 & 24 |
| Endless | 25+ | cycling | — | every 3 floors |

Key files:
- `src/data/balance.ts` — `MAP_CONFIG` (ROWS_PER_ACT=15, PRE_BOSS_ROW=13, BOSS_ROW=14), `SEGMENT_NAMES`, `SEGMENT_BOSS_FLOORS`, `FLOOR_TIMER`, `ENRAGE_SEGMENTS`, `ENEMY_TURN_DAMAGE_CAP`, `POST_ENCOUNTER_HEAL_CAP`, `DEATH_PENALTY`, `ENDLESS_BOSS_INTERVAL`, `MAX_FLOORS`
- `src/services/mapGenerator.ts` — `generateActMap(segment: 1|2|3|4, seed)`, uses `MAP_CONFIG` constants, derives floor from row using `6/ROWS_PER_ACT` ratio
- `src/services/floorManager.ts` — `getSegment()`, `getRegionForFloor()`, `BOSS_MAP`, `BOSS_POOL_BY_REGION`, `MINI_BOSS_POOL_BY_REGION`, `pickBossForFloor()`, `isBossFloor()`
- `src/services/gameFlowController.ts` — `retreatOrDelve` screen state, `generateActMap()` called on run start + after delve decision

---

## Design Decisions

- **D11:** Keep STS-style branching (7-8 nodes per act with 3-4 paths). Reuse existing map gen with reduced `ROWS_PER_ACT`.
- **D12:** Cut retreat-or-delve checkpoints entirely. 25-min runs don't need mid-run exits. Save/resume handles mid-session quits.
- **D13:** Gold economy still exists (~15-25g per regular combat, ~40-60g per elite/boss, ~400-800g per run total).
- **D17:** Keep all existing enemy sprites. Use 8-10 unique enemies per act (24-30 total). Remaining are stat variants.

## 3-Act Structure

### Act 1: The Shallows (Floors 1-4)

| Node Type | Count | Purpose |
|-----------|-------|---------|
| Combat (weak) | 3-4 | Card rewards, deck building |
| Mystery Room | 0-1 | Random events |
| Shop | 0-1 | Early purchases |
| Mini-Boss | 1 | Act gate, relic choice (1 of 3) |

Enemy pool: Simple, telegraphed. Cave Bat, Crystal Golem, Toxic Spore. Timer Wyrm mini-boss.
Philosophy: Players who path toward combat build faster. Quick Play viable. Charge optional.

### Act 2: The Depths (Floors 5-8)

| Node Type | Count | Purpose |
|-----------|-------|---------|
| Combat (medium) | 3-4 | Synergy testing |
| Elite | 1 | Quiz-focused, guaranteed relic |
| Rest Site | 1 | Heal / Study / Meditate |
| Shop/Mystery | 0-1 | Deck refinement |
| Boss | 1 | Quiz Phase at 50% HP |

Enemy pool: Shadow Mimic, Bone Collector. The Examiner elite. The Archivist boss.
Philosophy: Charging becomes necessary. Elite forces engagement.

### Act 3: The Archive (Floors 9-12)

| Node Type | Count | Purpose |
|-----------|-------|---------|
| Combat (hard) | 3-4 | Build-or-die |
| Elite | 1 | Final relic opportunity |
| Rest Site | 1 | Last heal/upgrade |
| Shop | 0-1 | Final purchases |
| Final Boss | 1 | The Curator — 2 Quiz Phases (66% and 33% HP) |

Enemy pool: The Scholar, The Nullifier elite, The Librarian elite. The Curator final boss.

### Total Run Metrics

| Metric | Value |
|--------|-------|
| Total rooms per path | ~16-18 |
| Combat encounters | ~10-12 |
| Boss/mini-boss fights | 2-3 |
| Shops | 1-2 |
| Rest sites | 1-2 |
| Estimated run time | 25-30 min |
| Quizzes per run | 45-70 |
