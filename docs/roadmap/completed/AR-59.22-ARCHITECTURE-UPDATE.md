## Data Flow Changes to Document

### Card Play Flow (v2)
Replace the old linear "play card → quiz → resolve" flow with the branching v2 flow:

```
Player taps card
  └─ Quick Play path:
       Card resolves at base power (no quiz, no multiplier)
       Chain resets if categoryL2 changes
  └─ Charge Play path:
       AP surcharge deducted
       Quiz panel slides in
       Player answers:
         ├─ Correct → card resolves at Charge multiplier × chain multiplier × (Surge ×2 if Surge turn)
         └─ Wrong   → card resolves at 0.7× base (partial effect, no fizzle)
       Chain advances or resets based on categoryL2 continuity
```

### Turn Rhythm (v2)
```
Turn 1: Normal
Turn 2: SURGE (first Surge)
Turn 3: Normal
Turn 4: Normal
Turn 5: SURGE
Turn 6: Normal
Turn 7: Normal
Turn 8: SURGE
...
```
Document that Surge turns double all Charge multipliers and are visually signaled one turn in advance.

### Relic Data Flow (v2)
```
Relic pickup event
  └─ 5-slot capacity check:
       ├─ Slot available → add relic, trigger passive effect registration
       └─ All slots full → open RelicSwapOverlay
            ├─ Sell existing relic → gold + slot freed → add new relic
            ├─ Swap existing relic → old relic removed → new relic added
            └─ Decline → new relic discarded
Relic reroll:
  └─ At relic offer node, player can spend gold to reroll offered relics (once per node)
```

### Run Structure Flow (v2)
```
Run Start → Act 1 (7–8 nodes) → Act 1 Boss
  → Act 2 (7–8 nodes) → Act 2 Boss
  → Act 3 (7–8 nodes) → Final Boss
  → Victory screen

At any point: defeat → run ends, stats recorded
No retreat-or-delve. No Endless mode.
```
