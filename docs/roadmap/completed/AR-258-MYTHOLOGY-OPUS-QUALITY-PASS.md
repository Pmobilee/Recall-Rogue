## Overview

Both mythology decks (Greek: 272 facts, Norse: 147+76 = ~223 facts) were generated by Sonnet workers for factual accuracy. The prose fields need an Opus quality pass to elevate writing quality, add narrative interweaving between sub-decks, and ensure stories read as connected arcs — not isolated trivia.

This is Phase 2 of the two-phase pipeline defined in `deck-master` SKILL.md.

---

## What Opus Rewrites

**REWRITE (prose quality):**
- `explanation` — richer context, cross-references to other sub-decks, narrative thread connections
- `wowFactor` — genuinely surprising, share-worthy, punchy
- `visualDescription` — vivid mnemonic scenes for pixel art cardback generation
- `statement` — clear, compelling, memorable

**DO NOT TOUCH (factual content):**
- `correctAnswer` — truth locked by Sonnet
- `quizQuestion` — already verified against source
- `distractors` — pool-based, already correct
- `difficulty`, `answerTypePoolId`, `chainThemeId`, `ageGroup` — structural
- `id`, `sourceUrl`, `sourceName`, `volatile` — metadata
