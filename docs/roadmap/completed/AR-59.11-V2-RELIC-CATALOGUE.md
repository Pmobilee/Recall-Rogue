# AR-59.11 — V2 Relic Catalogue

## Overview

**Goal:** Replace the current 50-relic catalogue (25 free starters + 25 unlockable) with the v2 design's 42-relic catalogue organized around 7 build archetypes. ~60% build-around relics, ~40% stat sticks.

**Dependencies:**
- AR-59.10 (5-slot relic system) — must exist before implementing
- AR-59.3 (Chain system) — required for Chain Reactor, Resonance Crystal, Tag Magnet, Echo Chamber, Prismatic Shard
- AR-59.4 (Surge system) — required for Time Warp
- AR-59.1 (Charge system) — required for Quicksilver Quill, Adrenaline Shard, Volatile Core, Crit Lens, Scholar's Crown, Insight Prism, Double Down, Domain Mastery Sigil, Memory Nexus, Capacitor, Overflow Gem

**Estimated Complexity:** Large — 42 relic definitions, new effect hooks, v1→v2 migration

**Source of Truth:** `docs/RESEARCH/recall-rogue-overhaul-v2.md` lines 315–550

---

## Design Decisions

### D14: Unlock Structure (25 Free + 17 Mastery Coin)
- **25 free at start:** All Commons (12) + selected Uncommons (13)
- **17 unlockable via Mastery Coins:** All Rares (12) + All Legendaries (2) + remaining Uncommons (3)
- Mastery Coin costs: Uncommon 35–45, Rare 50–60, Legendary 70+

### D16: Phase 1 Exclusions
Exclude relics referencing unbuilt mechanics. Launch with **38–40 functional relics**.
- **Excluded at launch:** `toxic_bloom` (requires multi-enemy encounter system, Phase 2)
- All other 41 relics are functional with v2's core systems (Charge, Chain, Surge)

## Build Archetypes

The v2 relic catalogue is organized around 7 build fantasies. Each archetype has 3 core relics and several supporting options.

| # | Archetype | Fantasy | Core Relics |
|---|-----------|---------|-------------|
| 1 | **Chain Master** | Long chains, exponential damage | Chain Reactor, Resonance Crystal, Prismatic Shard |
| 2 | **Speed Scholar** | Fast answers, massive bonuses | Quicksilver Quill, Adrenaline Shard, Time Warp |
| 3 | **Glass Cannon** | High risk, one-shot kills | Volatile Core, Reckless Resolve, Crit Lens |
| 4 | **Iron Fortress** | Unkillable, attrition wins | Aegis Stone, Thorn Crown, Regeneration Orb |
| 5 | **Poison Alchemist** | DoT stacking, indirect damage | Plague Flask, Festering Wound, (Toxic Bloom — Phase 2) |
| 6 | **Burst Master** | Save AP, nuke on one massive turn | Capacitor, Overflow Gem, Double Down |
| 7 | **Knowledge Engine** | Quiz mastery = combat mastery | Scholar's Crown, Memory Nexus, Insight Prism |

## Relic Drop Acquisition Rules (from v2 design)

Relics are acquired from:
- **Mini-boss kill:** Choose 1 of 3 relics (regular rarity weights)
- **Elite kill:** Guaranteed 1 relic drop (regular rarity weights)
- **Boss kill:** Choose 1 of 3 relics (boss rarity weights — better odds)
- **Regular combat:** 10% chance for 1 relic drop (regular rarity weights)

**At 5/5 slots:** Show all 5 held relics + new relic. Player must sell one to make room (or pass). Selling uses `RELIC_SELL_VALUE_*` gold constants. The "pass" option declines the new relic.

**Pity timer:** If no Uncommon+ relic has been offered in 4 consecutive drops, next drop is guaranteed Uncommon+. Timer resets on any Uncommon+ offer.

**Phase 1 filter:** Drop pool excludes any relic where `excludeFromPhase1 === true`.
