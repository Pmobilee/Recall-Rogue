# japanese_katakana — Expectations

## Scope
Complete katakana syllabary: 46 base characters + dakuten/handakuten variants + digraphs = 208 facts. Same structure as hiragana deck (forward + reverse pairs).

## Canonical Source
Standard Japanese Kana Chart.

## Sub-decks
None.

## Pool Inventory
| Pool | Facts | Synth |
|------|-------|-------|
| english_meanings | 208 | 0 |

## Quiz Template
`_fallback` exclusively (100%). No specialized katakana template.

## Quality Bar
- Forward: katakana character → romanization answer
- Reverse: romanization → katakana character
- Distractors match script type of the answer
- Katakana combinations (e.g., ヒョ, チョ, ミャ) correctly formed as single options

## Risk Areas
1. **POS-TELL (script mixing)** — 71 of 90 rows mix katakana characters and romaji in same option set; structurally identical issue to hiragana deck
2. **POOL-CONTAM** — single pool causes script-type mixing in distractors
3. **Visually similar characters** — ソ/ン, シ/ツ, ウ/ワ are notoriously similar; distractors should ideally include the visually similar pair (this is pedagogically valuable but may look like a distractor quality issue without context)
4. **Combination characters** — digraphs (ヒョ, チョ) are 2-character strings; distractor combinations must be valid katakana digraphs, not random character pairings
5. **No synthetic distractors** — 0 synth across the entire deck
