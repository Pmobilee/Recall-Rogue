# Competitor Analysis — Terra Gacha ASO (DD-V2-168)

## Key Competitors

### 1. Duolingo
- **Strength**: Streak system, daily reminders, gamification
- **Lesson**: Streak mechanics drive retention; we implement streaks + GAIA commentary
- **Weakness**: Repetitive exercise format; we offer mining variety

### 2. Mr. Driller / Dig Dug
- **Strength**: Satisfying dig mechanics, depth pressure
- **Lesson**: "One more layer" loop works; our O2 mechanic parallels their air mechanic
- **Weakness**: No educational content; we fill this gap

### 3. Idle Miner Tycoon
- **Strength**: Mining theme, casual audience, progression loops
- **Lesson**: Idle/active hybrid sessions; our 5-25 minute sessions serve similar need
- **Weakness**: No knowledge component; purely economic

### 4. Anki
- **Strength**: Gold standard spaced repetition, dedicated user base
- **Lesson**: SM-2 algorithm proven; we adopt and gamify it
- **Weakness**: Zero gamification, intimidating UI; we wrap learning in gameplay

### 5. Fossil Dig (various titles)
- **Strength**: Fossil collection hook, educational angle
- **Lesson**: Fossil revival is a strong motivator; we make it knowledge-gated
- **Weakness**: Typically shallow gameplay; we add roguelite depth
